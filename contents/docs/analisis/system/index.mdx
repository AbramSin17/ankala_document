## 1. Pendahuluan
## 1.1 Tujuan 
Dokumen ini bertujuan untuk mendefinisikan kebutuhan sistem yang diperlukan untuk pengembangan dan operasional website ankalajourney.site.

## 1.2 Lingkup 
Dokumen ini mencakup kebutuhan perangkat keras, perangkat lunak, jaringan, dan keamanan yang diperlukan untuk menjalankan website.

## 2. Kebutuhan Sistem
## 2.1 Kebutuhan Perangkat Keras

- Server
    - Prosesor: Minimal 4-core CPU (Intel Xeon atau setara).
    - RAM: Minimal 16 GB.
    - Penyimpanan: SSD dengan kapasitas minimal 500 GB.
    - Bandwidth: Koneksi internet dengan kecepatan minimal 100 Mbps.
- Pengguna
    - Komputer atau perangkat mobile dengan spesifikasi minimum:
    - Prosesor: Dual-core CPU.
    - RAM: Minimal 2 GB.
    - Penyimpanan: Minimal 1 GB ruang kosong.
    - Browser: Versi terbaru dari Chrome, Firefox, Safari, atau Edge.

## 2.2 Kebutuhan Perangkat Lunak
- Server
    - Sistem Operasi: Linux (Ubuntu, CentOS, atau Debian).
    - Web Server: Apache atau Nginx.
    - Database: MySQL atau PostgreSQL.
    - Bahasa Pemrograman: PHP dan Javascript
    - Framework: bootstrap.
- Pengguna
    - Browser web terbaru (Chrome, Firefox, Safari, atau Edge).
    - Koneksi internet yang stabil.

## 2.3 Kebutuhan Jaringan
- Server
    - IP statis untuk server agar dapat diakses secara konsisten.
    - Firewall untuk melindungi server dari akses tidak sah.
- Pengguna
    - Koneksi internet yang memadai untuk mengakses website tanpa gangguan.

## 2.4 Kebutuhan Keamanan
- Keamanan Data
    - Enkripsi data pengguna (misalnya, menggunakan HTTPS).
    - Perlindungan terhadap serangan DDoS.
    - Sistem autentikasi yang aman 
- Backup dan Pemulihan
    - Sistem backup otomatis untuk database dan file penting.
    - Rencana pemulihan bencana untuk memastikan kelangsungan operasional.

## 3. Kebutuhan Non-Fungsional
- Kinerja
    - Waktu respons halaman harus kurang dari 2 detik untuk 95% permintaan.
    - Kapasitas untuk menangani minimal 1000 pengguna secara bersamaan.
- Usability
    - Antarmuka pengguna harus intuitif dan mudah digunakan.
    - Website harus responsif dan dapat diakses di berbagai perangkat (desktop, tablet, smartphone).
- Kompatibilitas
    - Website harus kompatibel dengan browser utama (Chrome, Firefox, Safari, Edge) dan versi terbaru dari masing-masing.

## 4. Batasan
Pembayaran hanya dapat dilakukan dalam mata uang lokal.
