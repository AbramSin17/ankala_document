## Deskripsi Proyek
Proyek ini bertujuan untuk mengembangkan sebuah website bernama ankalajourney.site, yang menyediakan layanan jasa tourguide bagi wisatawan. Dengan meningkatnya minat wisatawan untuk menjelajahi destinasi baru, kebutuhan akan panduan lokal yang berpengalaman menjadi semakin penting. Website ini akan menjadi platform yang menghubungkan wisatawan dengan tour guide yang kompeten, memungkinkan pengguna untuk mencari, membandingkan, dan memesan jasa tour guide dengan mudah. Permasalahan yang ingin diselesaikan adalah kurangnya aksesibilitas informasi mengenai tourguide yang berkualitas dan kesulitan dalam proses pemesanan jasa tersebut. Dengan adanya website ini, diharapkan wisatawan dapat menemukan tourguide yang sesuai dengan kebutuhan mereka, serta meningkatkan pengalaman wisata secara keseluruhan.

## Teknologi yang Digunakan
Untuk mengembangkan website ankalajourney.site, berbagai teknologi akan digunakan, antara lain:
- Bahasa Pemrograman: PHP dan JavaScript untuk pengembangan backend.
- Framework: Bootstrap untuk styling dan responsif
- Database: MySQL untuk menyimpan data pengguna, tourguide, dan transaksi.
- Frontend: HTML, CSS, dan JavaScript untuk membangun antarmuka pengguna yang responsif dan interaktif.
- Hosting: rumahweb untuk melakukan hosting dan hostinger untuk melakukan pembelian domain

## Keterkaitan Mata Kuliah 
1. Pemrograman Web Lanjut
  Proyek ini sangat terkait dengan mata kuliah Pemrograman Web Lanjut, di mana mahasiswa akan menerapkan konsep-konsep lanjutan dalam pengembangan aplikasi web. Dalam proyek ini, mahasiswa akan menggunakan bahasa pemrograman seperti PHP untuk membangun backend website. Selain itu, penggunaan HTML, CSS, dan JavaScript untuk frontend juga merupakan bagian dari kurikulum yang diajarkan dalam mata kuliah ini.
2. Metode dan Model Pengembangan Perangkat Lunak
  Dalam pengembangan website ankalajourney.site, mahasiswa akan menerapkan berbagai metode dan model pengembangan perangkat lunak, seperti Agile atau Waterfall. Pemilihan model yang tepat akan membantu dalam merencanakan, mengorganisir, dan mengelola proses pengembangan secara efisien. Proyek ini memberikan kesempatan untuk memahami bagaimana model pengembangan mempengaruhi timeline, pengelolaan risiko, dan kolaborasi tim.
3. Basis Data
  Proyek ini melibatkan penggunaan basis data untuk menyimpan informasi penting, seperti data pengguna, informasi tourguide, dan transaksi pemesanan. Mata kuliah Basis Data memberikan pemahaman tentang desain, implementasi, dan pengelolaan basis data, yang sangat penting dalam memastikan integritas. Mahasiswa akan belajar bagaimana merancang skema basis data yang efisien dan melakukan query untuk mengambil data yang diperlukan.
4. Perancangan Antarmuka Pengguna
  Antarmuka pengguna (UI) yang baik sangat penting untuk meningkatkan pengalaman pengguna dalam menggunakan website. Dalam proyek ini, mahasiswa akan menerapkan prinsip-prinsip perancangan antarmuka pengguna yang telah dipelajari, seperti usability, aksesibilitas, dan desain responsif. Proyek ini memberikan kesempatan untuk merancang dan menguji antarmuka yang intuitif dan menarik bagi pengguna.
5. Praktik Analisis dan Desain Perangkat Lunak  
  Proyek ini juga berkaitan erat dengan mata kuliah Praktik Analisis dan Desain Perangkat Lunak, di mana mahasiswa akan melakukan analisis kebutuhan sistem dan merancang solusi perangkat lunak yang sesuai. Mahasiswa akan belajar bagaimana mengumpulkan dan menganalisis kebutuhan pengguna, serta merancang arsitektur sistem dan diagram alur yang jelas. Proyek ini memberikan pengalaman praktis dalam menerapkan teknik analisis dan desain yang telah dipelajari di kelas.
