## Deskripsi Proyek
Proyek ini bertujuan untuk mengembangkan sebuah website bernama ankalajourney.site, yang menyediakan layanan jasa tourguide bagi wisatawan. Dengan meningkatnya minat wisatawan untuk menjelajahi destinasi baru, kebutuhan akan panduan lokal yang berpengalaman menjadi semakin penting. Website ini akan menjadi platform yang menghubungkan wisatawan dengan tour guide yang kompeten, memungkinkan pengguna untuk mencari, membandingkan, dan memesan jasa tour guide dengan mudah. Permasalahan yang ingin diselesaikan adalah kurangnya aksesibilitas informasi mengenai tourguide yang berkualitas dan kesulitan dalam proses pemesanan jasa tersebut. Dengan adanya website ini, diharapkan wisatawan dapat menemukan tourguide yang sesuai dengan kebutuhan mereka, serta meningkatkan pengalaman wisata secara keseluruhan.

## Teknologi yang Digunakan
Untuk mengembangkan website ankalajourney.site, berbagai teknologi akan digunakan, antara lain:
- Bahasa Pemrograman: PHP dan JavaScript untuk pengembangan backend.
- Framework: Bootstrap untuk styling dan responsif
- Database: MySQL untuk menyimpan data pengguna, tourguide, dan transaksi.
- Frontend: HTML, CSS, dan JavaScript untuk membangun antarmuka pengguna yang responsif dan interaktif.
- Hosting: rumahweb untuk melakukan hosting dan hostinger untuk melakukan pembelian domain
